# zigtwitarchiveutil

A set of tools written in Zig to parse and analyze the data archive from X.com (formerly Twitter).

**Note:** This project is a work in progress.

## Requirements

- Zig 0.15.2

## Building

To build the project, run:

```sh
zig build
```

## Testing

To run the tests, run:

```sh
zig build test
```
